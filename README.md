# RS.Visual_Co

Sitio web portfolio para servicios de fotografia y video.

## Secciones
- Hero con propuesta de valor
- Portafolio
- Sobre mi
- Contacto

## Uso
- Abre `index.html` en tu navegador
- O usa un servidor local (por ejemplo Live Server)

## Estructura
- `index.html`: pagina principal
- `assets/`: estilos, scripts, imagenes y tipografias
- `components/`: navbar y footer
